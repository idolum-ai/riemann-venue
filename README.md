# riemann-venue

Computational and formal notes around changing the venue of the Riemann Hypothesis: cancellation, positivity, and spectral recentering.

This repository accompanies Daniel Rodriguez's essay sequence on primes, zeros, and the Riemann Hypothesis. It is not a proof of RH. It is a workspace for small formal/computational anchors around the third essay, "A Motion to Change Venue for the Riemann Hypothesis."

## Map

- `essay/` - essay drafts and companion text.
- `notes/` - focused mathematical notes.
- `notebooks/` - computational experiments and figures.
- `lean/` - Lean 4 / Mathlib formal anchors.
- `figures/` - generated images and diagrams.
- `references.bib` - bibliography seeds.

## First targets

1. Finite Mobius inversion on divisibility posets.
2. The finite gcd kernel `K(m,n) = gcd(m,n) / sqrt(mn)` and positivity experiments.
3. Tate/Gaussian vacuum notes for the completed zeta factor.
4. Weil functional notes as the missing positivity bracket.

## Caution

This repository is exploratory. It aims to make pieces of the essay inspectable, reproducible, and eventually formalizable, without claiming to prove the Riemann Hypothesis.
