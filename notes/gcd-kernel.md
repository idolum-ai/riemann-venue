# GCD kernel

Object:

```text
K(m,n) = gcd(m,n) / sqrt(mn)
```

Question: how much positivity is already visible at finite places through divisibility overlap?

Computational target:

- build finite truncations of `K`;
- inspect eigenvalues and conditioning;
- generate small figures;
- compare with normalized incidence/Mobius operators.

Interpretive role: arithmetic overlap represented as length.
