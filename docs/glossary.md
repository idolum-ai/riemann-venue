# Glossary

## Venue

The coordinate/categorical/spectral setting in which an arithmetic question is represented.

## No-leak condition

Essay language for the requirement that normalized error modes should not exhibit exponential instability after the right recentering.

## GCD kernel

The finite kernel `K(m,n)=gcd(m,n)/sqrt(mn)`, interpreted as divisibility overlap represented as length.

## Completed vacuum

Essay shorthand for the hoped-for completed object/state whose modular flow would implement arithmetic scale.
